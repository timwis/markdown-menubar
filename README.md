# Markdown Menubar

Take quick notes from the menu bar in markdown. Saves to a file
as you type. **Work in progress**.

![screenshot](http://i.imgur.com/PfXAHT5.png)

## Development
Install dependencies via
```bash
npm install
```

Run the application via
```bash
npm start
```
